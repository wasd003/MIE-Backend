## 1.2 注册

**post**  

**url**: `/auth/register`  

**body**  

```json
{
    "username": "Bob",
    "password": "123456"
}
```

**返回结果示例**  

```json
{
    "msg": "注册成功"
}
```

## 1.3 测试token有效性

**get**  

**url**: `/auth`  

**header**  

| key           | value          |
| ------------- | -------------- |
| Authorization | Bearer {token} |

注意Bearer和token之间有空格  

**返回结果示例**  

```json
{
    "id": 2
}
```
