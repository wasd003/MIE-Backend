# 1. 身份认证模块

## 1.1 登录

**post**  

**url**: `/auth/login`  

**body** 

```json
{
    "username": "Bob",
    "password": "123456"
}
```

**返回结果** 

| code  | msg              | data    |
| ----- | ---------------- | ------- |
| 0     | 成功登陆         | 见示例a |
| 10000 | 用户名或密码错误 | null    |
|       |                  |         |

**示例a**  

```json
{
	"token": "blahblahblah"
}
```

## 1.2 注册

**post**  

**url**: `/auth/register`  

**body**  

```json
{
    "username": "Bob",
    "password": "123456"
}
```

**返回结果**  

| code  | msg                  | data |
| ----- | -------------------- | ---- |
| 10001 | 密码太短或用户名为空 | null |
| 10002 | 用户已存在           | null |
| 0     | 注册成功             | null |
| 10003 | 注册失败             | null |



## 1.3 测试token有效性

**get**  

**url**: `/auth`  

**header**  

| key           | value          |
| ------------- | -------------- |
| Authorization | Bearer {token} |

注意Bearer和token之间有空格  

**返回结果示例**  

```json
{
    "id": 2
}
```
